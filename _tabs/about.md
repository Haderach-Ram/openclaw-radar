## OpenClaw Radar

Daily automated intelligence digest covering the OpenClaw ecosystem and competing AI agent frameworks.

### What's tracked

| Framework | Repo | Focus |
|-----------|------|-------|
| **OpenClaw** | [openclaw/openclaw](https://github.com/openclaw/openclaw) | Primary — full depth |
| **hermesagent** | [NousResearch/hermes-agent](https://github.com/NousResearch/hermes-agent) | Biggest competitor |
| **ZeroClaw** | [zeroclaw-labs/zeroclaw](https://github.com/zeroclaw-labs/zeroclaw) | Rust-based, privacy-first |
| **IronClaw** | [ironclaw-labs/ironclaw](https://github.com/ironclaw-labs/ironclaw) | nearai, "Reborn" architecture |

### Each digest contains
- 24h snapshot: new issues, closed, merged PRs, releases
- Per-repo breakdown with hot threads
- Our own filed issues: status tracker
- Posted daily at ~07:45 IST

### Inspiration & credits
This project is inspired by and built on the designs of:
- [duanyytop/agents-radar](https://github.com/duanyytop/agents-radar) — the gold standard for AI ecosystem digest repos. Design philosophy, GitHub Pages + RSS approach, and dual Issue+file storage pattern all borrowed from here.
- [ivanweng2077/big_model_radar](https://github.com/ivanweng2077/big_model_radar) — original inspiration for the competitive monitoring concept.
